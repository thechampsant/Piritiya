# Phase 1 Implementation Checklist

Use this checklist to track your progress through Phase 1.

## Day 1-2: AWS Infrastructure Setup

### AWS Account Setup
- [ ] AWS account created/accessed
- [ ] AWS CLI installed (`aws --version`)
- [ ] AWS credentials configured (`aws configure`)
- [ ] Credentials tested (`aws sts get-caller-identity`)

### Bedrock Access
- [ ] Navigate to AWS Bedrock console
- [ ] Request model access
- [ ] Enable Claude 3.5 Sonnet
- [ ] Enable Titan Embeddings G1 - Text
- [ ] Verify access granted

### S3 Buckets
- [ ] Create `piritiya-data` bucket
- [ ] Create `piritiya-knowledge-base` bucket
- [ ] Verify buckets exist (`aws s3 ls`)
- [ ] Test bucket access

### DynamoDB Tables
- [ ] Run `python scripts/create_dynamodb_tables.py`
- [ ] Verify Farmers table created
- [ ] Verify NISARData table created
- [ ] Verify CropRecommendations table created
- [ ] Verify Consultations table created
- [ ] Check tables in AWS Console

### IAM Roles
- [ ] Lambda execution role will be created by deploy script
- [ ] Verify permissions include DynamoDB, S3, CloudWatch

Build a React PWA called Piritiya — a voice-first farming assistant for farmers in Uttar Pradesh, India.

I have already provided the design system in a folder called piritiya-design-system/ which lives outside the frontend folder. Use relative paths to import from it. Use piritiya-design-system/DESIGN_SYSTEM.md as your complete reference. Do not invent new styles, colors, or components — everything you need is already defined.

If the bundler blocks imports outside src/, add this alias to vite.config.js:

resolve: {
  alias: { '@ds': path.resolve(__dirname, '../piritiya-design-system/src') }
}

Then import as: import { colors } from '@ds/tokens'

LangSheet bottom sheet:
- Triggered by any LangToggle tap
- Slides up with slideUp animation (cubic-bezier(0.32,0.72,0,1))
- Blurred dark backdrop, tap to dismiss
- 2-column grid of language tiles
- Each tile: script name large + roman name small + Instant badge (hi/en) or Slower badge (others)
- Selected tile: green border + green script text + Check icon
- Selecting a batch language (non hi/en): show 4-second warning toast

---

Language system:
- Default language: "hi" (Hindi)
- Use getTranslation(lang) for all UI strings — never hardcode English text
- Use toLocalNum(n, lang) for all numbers displayed in UI
- 8 supported languages: hi, en, bn, gu, kn, ml, ta, te
- Non-hi/en falls back to English UI strings until full translations added

---

Voice orb behaviour:
- Idle: tricolour gradient (saffron top, white mid, green bottom), 4s slow pulse, navy #000080 mic icon at 80% opacity
- Tap to listen: ripple rings animate outward, mic fades out, "Listening..." text pulses below
- After 2.6s: pick a random sample query, navigate to chat screen, send as message
- 20% chance of failure: show yellow voice error toast with fallback to type mode
- Voice disabled in settings: tap shows error toast immediately

Strict design rules — never break these:
- All text: #1a2010 or rgba(20,30,16,x) — never white text on the light background
- Primary action color: #138808 (India flag green)
- Card surfaces: rgba(255,255,255,0.75), border 1px solid rgba(0,0,0,0.07), shadow 0 2px 12px rgba(0,0,0,0.06)
- Border radius: cards 20px, pills 100px, small 8–12px
- All tappable elements: minHeight 44px
- No emoji anywhere — only icons from piritiya-design-system/src/icons
- AmbientBg on every screen at zIndex 0
- Animations use keyframes from animation.keyframes — inject once at app root via globalStyles
- Inputs: background rgba(0,0,0,0.06), border rgba(0,0,0,0.12), text #1a2010

AWS integration (mock data for now, structure for real):
- Amazon Transcribe: voice-to-text (real-time hi/en, batch for bn/gu/kn/ml/ta/te)
- Amazon Polly: text-to-speech (Kajal Neural for Hindi, Aditi for English)
- Amazon Bedrock Claude: chat responses
- Amazon Translate: message translation
- IndexedDB via idb library: offline message queue
